Hello{{Name}}!

Here are the amazing things you can do with the app!

  • Type in http://localhost:3000/greet/(anyname) and we will greet you by that name! Example: http://localhost:3000/greet/Cornelius
  • Do a post request with a JSON object to http://localhost:3000/greet/ using superagent (Example: superagent localhost:3000/greet post '{"name": "Cornelius"}'), and you will get a JSON string back!
